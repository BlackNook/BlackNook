EZGradientGenerator

EZGradientGenerator is a gradient management software.

It can help you to use the gradient effect when you save/manage the project.

It can be converted to code for development use.

Function is as follows

. Managed gradient style groups.

. The gradient object can be edited directly.

. You can view the gradient drawing source code.

. The source code can be directly cut (copied) and used by XCode.

. Support macOS-ObjectC/ Swift code output.

. Support iOS-ObjectC/ Swift code output.
