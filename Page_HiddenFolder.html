Hidden Folder

HiddenFolder is a working folder hiding and management tool.

It helps you manage folders and support folder hiding/displaying.

Features:

. Folder management, support dragging new / arranging functions.

. Folder hiding, support to hide or display the working folder.

. Quick Links, Support double-click or right-click menu to open the folder using Finder.

. Support startup security code.

. Support for opening hidden state folder in the Finder.

. Support conversion system hidden folder.
