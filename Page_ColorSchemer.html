ColorSchemer

ColorSchemer is a color management software.

It can help you save and manage the color effects you need for your project.

It can be converted to code for development use.

Function is as follows

. Manage color style groups.

. The system palette can be used to create colors.

. Color can be created using the color wheel.

. You can use a pattern file to create a color.

. The source code can be directly cut (copied) and used by XCode.

. Support NSObject (ObjectC/Swift) / UIColor (ObjectC / Swift) / CGColor / Java / RGB..

. A color description text file can be output.
