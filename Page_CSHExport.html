CSHExport

CSHExport is a software for the Photoshop Custom Shape file conversion.

Can help you convert vector graphics in CSH files into code/SVG files/PNG graphics.

For development use.

Function is as follows

. Load Photoshop Custom Shape files.

. The vector path can be viewed.

. Can view the source code.

. The source code can be directly cut (copied) and used by XCode.

. Support for ObjectC-NSBezierPath/UIBezierPath/CGPathRef code output.

. Support Swift-NSBezierPath/UIBezierPath/CGPathRef code output.

. Support for Java code output.

. Support for SVG vector graphics output.

. Support for PNG graphics output.
